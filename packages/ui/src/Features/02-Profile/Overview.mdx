## Profile Fields

### Required (30%)
- Name
- Email (@buffalo.edu)
- Graduation year

- Avatar photo
- Handle (@username)
- Major/program

### Complete (70%)

## Privacy & Security

### Campus Isolation
- Only UB students see profiles
- Email verification required
- No external access

### Privacy Controls

### Data Ownership

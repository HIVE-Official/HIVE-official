# 01-Onboarding

**User signup and profile creation flow**

## Purpose

The onboarding feature guides new users through creating their account, setting up their profile, and selecting their interests. The goal is to get users to "aha moment" as quickly as possible.

## Components in this Feature

- **Onboarding Wizard** - Multi-step flow for account creation
- **Interest Selector** - Choose topics and communities
- **Welcome Mat** - First-time user welcome screen
- **Completion Psychology Enhancer** - Progress motivation and encouragement
- **Hive Progress** - Visual progress indicators

## Key Flows

1. **Email Verification** - @buffalo.edu email required
2. **Profile Setup** - Name, avatar, bio
3. **Interest Selection** - Choose at least 3 interests
4. **Space Recommendations** - Suggested communities to join

## Design Principles

- **Fast** - Complete in < 2 minutes
- **Mobile-first** - Most users onboard on phones
- **Progressive** - Show value before asking for commitment
- **Skippable** - Allow users to explore before completing

## Success Metrics

- Time to complete: < 90 seconds
- Completion rate: > 80%
- Interest selection: 3-5 interests average
- Space joins: 2-3 spaces on first session

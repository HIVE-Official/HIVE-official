# Interaction Contracts

States
- Idle, Hover, Focus-visible, Active, Disabled, Loading, Error, Selected/Current.

Keyboard
- Generic: Tab/Shift+Tab, Space/Enter.
- Menus: Arrow keys, Home/End, typeahead, Escape.
- Tabs: Arrow keys within tablist.
- Dialog/Sheet: Escape to close; focus trap; return focus.

ARIA
- Inputs: labeled by `Label htmlFor`; helper/error via `aria-describedby`.
- Tooltip: `role="tooltip"` with labeled trigger when needed.
- Progress/Spinner: `role="progressbar"` with `aria-valuenow` when determinate.

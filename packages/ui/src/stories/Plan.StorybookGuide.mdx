# Storybook Guide

- Use structured titles: `Group[/Domain]/Component[/Variant]`.
- Top groups: Plan, Foundation, Brand, Atoms, Molecules, Organisms, Layouts, Templates, Pages, Reference.
- Domain subgroups under Organisms: Spaces, Profile, Onboarding, HiveLab.
- Default layout is `padded`; use `fullscreen` for shells/layouts only.
- Add `tags: ['autodocs']` on component stories to enable docs.

See `packages/ui/src/stories/README.md` for conventions and checklist.

## ğŸ¨ Color Philosophy

### Brand Gold (#FFD700)
- Primary action color
- Highlights important elements
- Creates "premium" feeling

- Reduces eye strain

- Better for late-night use (70% of campus usage)
- Conserves battery on OLED screens

- **Success**: Green (`#10B981`)
- **Warning**: Amber (`#F59E0B`)
- **Error**: Red (`#EF4444`)
- **Info**: Blue (`#3B82F6`)

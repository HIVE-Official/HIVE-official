# Weekly Audit Checklist — UI/UX/IA

Owner: Design System Guild • Attendees: A11y Lead, Feature Leads

- IA compliance
  - Page section order matches guardrails
  - Only allowed molecules per page are used
- A11y
  - Focus visible; keyboard paths complete; SR labels/roles present
  - Contrast AA+ on dark; motion respects reduced‑motion
- Brand & tokens
  - Gold usage quota (≤3 per viewport); no gold‑on‑gold
  - No token drift (colors, radius, spacing, elevation, motion)
- Motion ranges
  - Micro 100–140ms; macro 160–220ms; fallbacks present
- Performance
  - Story skeletons < 100ms; LCP < 2.5s in app exemplars
- Measurement
  - Event taxonomy implemented for new flows; guardrails monitored
- RFC hygiene
  - New patterns submitted via 10‑minute RFC; Canon updated when approved
- Debt
  - Deprecated variants scheduled for removal; docs/stories updated
